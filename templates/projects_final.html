AI-Powered Portfolio

A self-measuring portfolio that turns job applications into a trackable funnel.

Problem

Job applications usually disappear into a black box. I had no visibility into whether recruiters were opening my portfolio, revisiting it, or showing any real signal after I applied.

Solution

Built a FastAPI + PostgreSQL portfolio with application-specific ref links, recruiter visit logging, private analytics, time-on-site tracking, and a Groq-powered insight layer that surfaces patterns from real job-search data.

Impact

This project changed my portfolio from a brochure into a measurement system. I can now identify viewed applications, repeated visits, likely high-intent recruiters, and where to focus follow-ups instead of relying on guesswork.

My Role

End-to-end owner: product framing, data model design, backend APIs, Jinja frontend, dashboard logic, SQL analysis, GA4 event integration, cache strategy, security hardening, and deployment workflow.

Marketing Campaign Performance & Budget Optimization Dashboard

Built a decision-focused dashboard to diagnose funnel inefficiencies and optimize budget allocation across multi-language ad campaigns.

Problem

Marketing campaigns across regions were generating inconsistent results, with varying costs per subscription (CPS). There was no clear visibility into where users dropped off in the funnel or which campaigns deserved more budget.

Solution

Built an interactive Power BI dashboard analyzing the full funnel (Impressions → Clicks → Installs → Activations → Subscriptions). Identified bottlenecks using conversion rates and conditional formatting, and connected CPS/CTR with spend analysis to guide budget allocation decisions.

Impact

Enabled clear identification of scalable vs inefficient campaigns, surfaced actionable fixes (CTR issues in Marathi, post-click conversion gaps in Malayalam), and supported budget allocation simulations to target a blended CPS of ~₹280.

My Role

Designed the analytical framework, built the full Power BI dashboard (data modeling, DAX measures, and visuals), and translated insights into practical growth actions focused on real decision-making.

Sales Analysis Dashboard

A Power BI dashboard built to surface sales, profit, customer, and regional performance in one decision layer.

Impact

This project demonstrates the fundamentals I want hiring managers to see in analyst roles: KPI selection, slicing performance by business dimension, and turning operational data into a report someone could actually use.

My Role
